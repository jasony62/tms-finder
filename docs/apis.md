| API               | Method | 说明     |
| ----------------- | ------ | -------- |
| file/upload/mkdir | GET    | 新建目录 |
| file/upload/rmdir | GET    | 删除目录 |

# 新建目录（file/upload/mkdir）

## 查询参数

| 参数 | 说明         | 默认值 |
| ---- | ------------ | ------ |
| dir  | 要新建的目录 | -      |

# 删除目录（file/upload/rmdir）

## 查询参数

| 参数 | 说明         | 默认值 |
| ---- | ------------ | ------ |
| dir  | 要删除的目录 | -      |
